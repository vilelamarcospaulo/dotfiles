# codemonkey

Initiative management for Claude Code.

```
/codemonkey:start                      Start a new initiative (guided intake)
/codemonkey:load {name}                Load an initiative and get a briefing
/codemonkey:list                       List active initiatives
/codemonkey:list --include-finished    List all initiatives
/codemonkey:status {name}              Full status report for an initiative
/codemonkey:include                    Add context to the current initiative
/codemonkey:include {name}             Add context to a specific initiative
/codemonkey:sync                       Update current initiative state after a session
/codemonkey:sync {name}               Update a specific initiative
```
